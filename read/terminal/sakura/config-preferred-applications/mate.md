# Mate


## 設定指令


### set

執行下面指令

``` sh
gsettings set org.mate.applications-terminal exec 'sakura'
```


### get

執行下面指令

``` sh
gsettings get org.mate.applications-terminal exec
```

顯示

```
'sakura'
```


### reset

執行下面指令

``` sh
gsettings reset org.mate.applications-terminal exec
```
